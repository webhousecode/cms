# F161.3 — Delt designet capabilities-renderer i @broberg/cms-chat-client

> Parent: F161. Momentum fra broberg-capabilities-panelet (2026-07-11).

## Motivation

Chat'ens "Hvad kan du?"-svar renderes som flad markdown i cms-admin. broberg.ai har nu et lækkert designet capabilities-panel (kort + ikoner + chips). Løft det designede layout ind i den delte `@broberg/cms-chat-client`, så ALLE tre consumers (cms-admin, broberg, sanne) renderer ens i stedet for flad markdown — én kilde, ingen drift.

## Scope

- Tilføj en framework-agnostisk designet-capabilities-renderer (eller data+CSS-kontrakt + lille vanilla-renderer) til pakken.
- Adoptér den i cms-admins egen chat for capabilities-handlingen.
- Konvergér broberg's bespoke version onto den delte (fjern drift-kopi i admin-chat.tsx).

## Non-goals

- Ændre capabilities-INDHOLDSkilden (stadig CMS-motor / kurateret liste).
- Sanne-adoption (egen motor, separat).

## Arkitektur-skitse

Panelet er i dag bespoke i broberg's `admin-chat.tsx` (CAPS-data + CAPS_CSS + Capabilities-komponent). Løft CSS + render ind i pakken — React/Preact-agnostisk (eksportér en render-funktion ELLER data+CSS + vanilla-renderer), keyed off capabilities-markdown/struktureret data. cms-admin renderer det for capabilities-quick-action. Udgiv via OIDC.
