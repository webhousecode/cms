# F161.4 — Dead-code sweep (code-audit)

> Parent: F161.

## Motivation

Akkumuleret ubrugt kode (filer/exports/deps) i cms/cms-admin øger bundle + kognitiv byrde. `scripts/code-audit.sh` findes allerede til at finde det.

## Scope

- Kør `scripts/code-audit.sh`, triagér fund i (bekræftet dødt / tvetydigt / falsk-positiv).
- Slet KUN bekræftet-dødt (filer/exports/ubrugte deps), kirurgisk, med verifikation pr. sletning.
- typecheck + fuld test-suite grøn efter.
- List tvetydige/grænse-tilfælde til review i stedet for at slette.

## Non-goals

- Refaktorering af levende kode.
- Adfærdsændringer.
- Sletning af noget load-bearing (Rule 3: nævn, slet ikke, hvad du er i tvivl om).

## Arkitektur-skitse

Ren oprydning. Audit → triage → surgical delete af bekræftet-dødt → verificér grøn. Følg surgical-changes-reglen.

## Afhængigheder

scripts/code-audit.sh.
